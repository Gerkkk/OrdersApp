# Frontend

## Запуск

При желании можно заменить содержимое config.js

```javascript
const config = {
    apiBaseUrl: 'http://localhost:8081',
};


export default config;
```
Для локального запуска при текущих настройках бэкенда нужно использовать уже созданный конфиг.

```bash
    npm run build
```
